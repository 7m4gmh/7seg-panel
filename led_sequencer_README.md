# led_sequencer

`led_sequencer.py` はリポジトリ内の `config.json` を読み、7セグメント表示の点灯シーケンスを実行する簡易スクリプトです。

## 概要
- 各桁について、セグメント A → G を順に 0.5 秒おきに点灯します。
- その後、各桁ごとにその桁のみ全セグメントを 0.5 秒間点灯（他桁は消灯）します。
- 最後の桁が終わると最初に戻り、ループします。

## 必要条件
- Python 3.8+（型注釈の互換性のため）
- `config.json` がスクリプトと同じディレクトリに配置されていること（本リポジトリのルート `7seg-panel/config.json` を参照します）。

## 使い方

```bash
# デフォルト（delay=0.5s, 自動でエミュレータ構成を選択）
python3 led_sequencer.py

# レイアウトを指定する例（config.json のキー名を指定）
python3 led_sequencer.py --layout emulator-24x4

# ディレイを変更する例（秒単位）
python3 led_sequencer.py --delay 0.3
```

## 出力
- 実行中は標準出力にどの桁／セグメントを点灯したかがタイムスタンプ付きで表示されます。実機やエミュレータに接続する場合は、この部分を適宜置き換えてハード制御コードを追加してください。
